This readme is just to give an overview of Python files. There are six Python files in the folder 'code', 
below are there names and short description:

1. Generic_Functions - Contains common functions used in other Python files.
2. Exploratory_Data_Analysis - Computes distribution, plot and word clouds related to data.
3. NB_LR - Code for establishing baseline.
4. Logistic_Regression - Contains experiments of Logistic Regression with different problem transformation methods.
5. Multinomial_Naive_Bayes - Contains experiments of Multinomial Naive Bayes with different problem transformation methods.
6. Decision_Trees - Contains experiments of Decision Trees with different problem transformation methods.
